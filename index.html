    Grimoire of Arcturus


    Description

    Telling the story of a world that has been destroyed and filled with monsters, the Grimoire of Arcturus is a magical book that can cast spells to defeat enemies during the journey to find the obelisks. Players are required to find and activate four obelisks, so that the gate to the Tower of Arcturus can be opened. During their search, players will be accompanied by a companion named Wy-E, who helps illuminate the darkness of the adventure.


    Eazy Golf


    Description

    Rock Paper Scissor


    Description

    Rock Paper Scissors (RPS) is a game based on the traditional game but with more attractive character designs. As players, we will face opponents controlled by bots that have their own strategic selection mechanisms. Both players will compete until one player's three health points are depleted.


    Sinner Run!


    Description

    Sinner Run is an endless runner game where players focus on avoiding obstacles as the character's running speed increases over time. In this game, players survive as long as possible while passing through obstacles to achieve the highest score. There are also coins that can be collected and exchanged for skins in the shop. Additionally, there are buffs that players can pick up, providing temporary invincibility against obstacles.


    Dental Education


    Description

    Dental Education is an interactive game about proper tooth brushing techniques. There are no objectives required in this game; it simply involves user interaction with the available mini-games, such as selecting the correct technique for brushing specific parts of the teeth. There is also an interactive book containing information about dental health.

    Gigi Edukasi


    Description

    Gigi Edukasi is a tooth brushing simulation game designed for children. The objective of this game is quite simple: clean all the dirt off the teeth before time runs out and proceed to the next level. There are videos using sign language aimed at deaf children, allowing them to understand the purpose of the game.
